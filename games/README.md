# 實現終端查看 NBA 賽程與比賽結果

## Step0 environment setup:

下載 nba_api package

```bash
pip install nba_api
```

## 功能預覽:
執行 __main__.py
```bash
python __main__.py
```




輸入想查詢的 年-月-日(yyyy-mm-dd)

即可查詢該日期的比賽和結果

:warning:ps. 19XX 年之前的數據不完整，古老球隊名稱也不齊全，可能會出錯
